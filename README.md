```
查看文件列表命令
ls	# 查看当前目录的文件列表
ls -a	# 查看当前目录所有文件列表
ls -l	# 查看当前目录的文件详细信息   ls -la	# 查看当前目录所有文件详细信息 ls -ld	# 查看当前目录本身的详细信息   ls -Rm /boot # 以递归紧凑方式查看/boot 目录 ls -lt	# 按时间排序 (从新到旧)
ls -ltr	# 按时间反向排序 (从旧到新)
ls -lS	# 按文件大小排序 (从大到小)
```
